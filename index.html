Cara Penggunaan:

  1. Judul Data (Opsional): Masukkan judul untuk data kelompok Anda. Jika tidak diisi, akan menggunakan "Untitled" secara otomatis.
  2. Masukkan jumlah kelompok yang diinginkan
  3. Tambahkan peserta dengan tombol "Tambah Peserta" atau tekan Enter setelah mengetik nama
  4. Pilih jenis kelamin untuk setiap peserta (Putra/Putri) - opsional
  5. Pengaturan Gender (opsional):
    • Pisahkan kelompok berdasarkan gender (Putra dan Putri terpisah)
    • Aktifkan minimal Putra/Putri per kelompok untuk distribusi seimbang
  6. Klik "Buat Kelompok" untuk membagi secara otomatis dan acak
  7. Setelah hasil muncul:
    • Edit Nama Kelompok: Klik pada input field nama kelompok di setiap kartu kelompok untuk mengubah nama kelompok sesuai keinginan
    • Klik "Update Kelompok" (tombol akan berubah otomatis dari "Buat Kelompok") untuk memperbarui kelompok yang sudah ada dengan perubahan peserta
    • Klik "Salin Hasil" untuk menyalin hasil ke clipboard
    • Klik "Acak Ulang" untuk membuat pembagian baru secara acak
    • Klik "Simpan sebagai Gambar" untuk menyimpan hasil sebagai gambar PNG (untuk tampilan/print)
    • Klik "Simpan sebagai TXT" untuk menyimpan hasil sebagai file teks yang bisa diimport kembali
  8. Import Data:
    • Gunakan tombol "Import Data Kelompok (TXT)" untuk memuat data dari file TXT yang sebelumnya disimpan
    • File TXT yang diimport akan memuat: judul data, nama kelompok, dan semua peserta dengan gender
    • Setelah import, Anda bisa langsung melihat hasil atau melakukan perubahan
  9. Klik "Reset Semua" untuk menghapus semua data (peserta, kelompok, dan pengaturan) dan memulai dari awal

Cara Penggunaan:

  1. Masukkan judul data (opsional) untuk mengidentifikasi pembagian kelompok Anda
  2. Tentukan jumlah kelompok yang ingin dibuat
  3. Tambahkan peserta dengan dua cara:
    • Gunakan tombol "Tambah Peserta" atau tekan Enter setelah mengetik nama
    • Gunakan fitur "Tambah Peserta Secara Bulk" untuk menambahkan banyak peserta sekaligus (satu nama per baris)
  4. Pilih jenis kelamin untuk setiap peserta (Putra/Putri) - opsional
  5. Konfigurasi pengaturan gender (opsional):
    • Aktifkan pemisahan berdasarkan gender untuk memisahkan Putra dan Putri
    • Atur jumlah minimal Putra/Putri per kelompok untuk distribusi yang seimbang
  6. Klik tombol "Buat Kelompok" untuk memulai pembagian otomatis secara acak
  7. Setelah hasil muncul, Anda dapat:
    • Edit nama kelompok dengan mengklik field nama di setiap kartu kelompok
    • Klik "Update Kelompok" untuk menyimpan perubahan nama atau peserta
    • Klik "Salin Hasil" untuk menyalin hasil ke clipboard
    • Klik "Acak Ulang" untuk membuat pembagian baru secara acak
    • Klik "Simpan sebagai Gambar" untuk menyimpan hasil sebagai file PNG
    • Klik "Simpan sebagai TXT" untuk menyimpan hasil sebagai file teks yang dapat diimport kembali
  8. Untuk memuat data yang sudah disimpan:
    • Klik tombol "Import Data Kelompok (TXT)" dan pilih file TXT yang sebelumnya disimpan
    • File TXT berisi: judul data, nama kelompok, dan semua peserta beserta gender
    • Setelah import, hasil akan langsung ditampilkan dan Anda dapat melakukan perubahan
  9. Klik tombol "Reset Semua" untuk menghapus semua data dan memulai dari awal

Cara Penggunaan:

  1. Masukkan jumlah kelompok yang diinginkan
  2. Tambahkan peserta dengan tombol "Tambah Peserta" atau tekan Enter setelah mengetik nama
  3. Pilih jenis kelamin untuk setiap peserta (Putra/Putri) - opsional
  4. Pengaturan Gender (opsional):
    • Pisahkan kelompok berdasarkan gender (Putra dan Putri terpisah)
    • Aktifkan minimal Putra/Putri per kelompok untuk distribusi seimbang
  5. Klik "Buat Kelompok" untuk membagi secara otomatis dan acak
  6. Setelah hasil muncul:
    • Klik "Salin Hasil" untuk menyalin ke clipboard
    • Klik "Acak Ulang" untuk membuat pembagian baru
    • Klik "Simpan Kelompok (TXT)" untuk menyimpan ke file
  7. Import Data: Gunakan tombol "Import Data Kelompok (TXT)" untuk memuat data dari file TXT
  8. Klik "Reset Semua" untuk menghapus semua data dan memulai dari awal
